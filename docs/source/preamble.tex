\inrTitle{TSP: Python package to facilitate preparation of input files}
\inrProject{}{}
\inrDate{November 2013}

\inrAbstract{%
Preparation of input files for computer codes is an essential and routine task
in nuclear reactor design. A concept to simplify this task is proposed and
software, developed to support this concept, is described in this
report.
}
